# Business-Sales-Profit-Analysis-Dashboard
This project is an Excel-based dashboard created to analyze and understand business sales data. It focuses on key metrics like revenue, expenses, and profit, making it easier to track overall performance and identify trends.

# Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Graphs
- Excel formulas

# Key Highlights
- Clear view of overall sales and profit
- Analysis based on region and product category
- Comparison of different business segments
- Simple and easy-to-understand visual representation of data

# Dataset
The dataset contains transaction-level details such as date, region, category, sales amount, expenses, and profit.

# Purpose
The main aim of this project was to practice Excel skills and learn how to convert raw data into meaningful insights that can support better decision-making.
